# UAC Care System — Capacity & Load Analytics

A Streamlit dashboard for analyzing the U.S. HHS Unaccompanied Alien 
Children (UAC) program care pipeline from January 2023 to December 2025.

## Project Overview
This dashboard provides real-time analytics on:
- Total system load across CBP and HHS
- Net intake pressure and backlog trends
- Capacity stress identification
- Monthly care load breakdowns

## Tech Stack
- Python 3.x
- Streamlit
- Pandas / NumPy
- Plotly
